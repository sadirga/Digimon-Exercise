# Digimon-Exercise
Scrapping data from digidb.io and insert the data into SQL table  
[![logo.png](https://i.postimg.cc/SK2NcQ9J/logo.png)](https://postimg.cc/mc4sss8s)

## Langkah Pertama
 - Import library mySQL connector dan library untuk scrapping
  ```python
  import mysql.connector
  from bs4 import BeautifulSoup
  import requests
  ```

## Langkah Kedua
  - Amati page source web digidb.io digimon/list untuk melihat letak nama-nama digimon, url image dan stats digimon nya.
  - Setelah itu, barulah kita mulai scrap untuk mengambil data yang ktia butuhkan
  ```python
  link = 'http://digidb.io/digimon-list/'
  url = requests.get(link)
  soup = BeautifulSoup(url.content, 'html.parser')

  data = soup.findAll('td')                         # Di dalam tags 'td' terdapat data-data yang kita butuhkan
  digimons = []                                     
  for i in range(0,len(data)):                      # For loop disini untuk store data-data digimon dari web digidb.io ke dalam list
      digimons.append(data[i].text.strip())         # Menggunakan strip untuk menghilangkan spasi sebelum atau sesudah jika ada pada string
  print(digimons)                                   # Cetak untuk cek hasilnya
  ```

# Langkah Kelima
  - Setelah list image link dibuat, maka sekarang perlu membuat tabel SQL
  - Untuk Catatan, disini saya sudah terlebih dahulu membuat database kosong via MySQL
  ```python
  # Create Table SQL
    db = {
      'user' : 'username',                  # input username
      'password' : '********',              # input password
      'host' : 'localhost',
      'database' : 'digimon'
  }

  conn = mysql.connector.connect(**db)
  kursor = conn.cursor()

  q = """ CREATE TABLE digimon_list(        
      No SmallInt NOT NULL AUTO_INCREMENT,   # 1
      Nama_Digimon varchar(30),              # 2
      ImgLink varchar(50),                   # 3
      Stage char(20),                        # 4
      Type char(20),                         # 5
      Attribute char(20),                    # 6
      Memory TinyInt,                        # 7
      Equip_slots TinyInt,                   # 8
      HP SMALLINT,                           # 9
      SP SMALLINT,                           # 10
      Atk SMALLINT,                          # 11
      Def SMALLINT,                          # 12
      Intl SMALLINT,                         # 13
      Speed SMALLINT,                        # 14
      Primary Key(No)                        
  )
  """
  kursor.execute(q)

  print('Sukses Membuat Table')
  ```
